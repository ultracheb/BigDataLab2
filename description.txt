Процесс взаимодейсвия со Scala очень похож на Rx фреймворк для c# или java, код представляет из себя цепь преобразований над объектами. В нашем случае мы сначала делим строчку на массив по сепаратору, потом формируем объект из второго и третьего элемента строчки(id вершины и вес дуги, которая входит в вершину), затем операцией reduceByKey складываем все значения по ключу(все веса вершины по id), дальше из пары id-сумма формируем строчку "id сумма" для каждой вершины
